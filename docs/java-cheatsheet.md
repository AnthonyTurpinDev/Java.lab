# Java Cheatsheet

## Types de base
- int, double, boolean, char, String

## Boucles
- for, while, do-while

## Collections
- ArrayList, HashMap, HashSet

## Complexités
- Bubble sort : O(n^2)
- Recherche linéaire : O(n)
- Recherche dichotomique : O(log n)

## Conseils
- Préférez les noms explicites
- Mesurez les performances avec `System.nanoTime()`
- Testez d’abord dans `Playground.java`
