# java.lab

Projet Java de laboratoire destiné à l’apprentissage structuré du langage Java, avec une progression claire entre les bases, les exercices d’algorithme, l’orienté objet et les structures de données.

## Organisation du dépôt

- src/main/java/lab/sandbox/Playground.java : espace de test personnel et d’expérimentation rapide
- src/main/java/lab/utils/Printer.java : utilitaire d’affichage
- src/main/java/lab/utils/Timer.java : utilitaire de mesure de temps
- src/main/java/lab/exercises/algo/condition : exercices sur les conditions et les décisions
- src/main/java/lab/exercises/algo/loop : exercices sur les boucles et l’itération
- src/main/java/lab/exercises/oop : exercices orientés objet
- src/main/java/lab/exercises/structures : exercices sur les tableaux et structures de données
- archive/legacy : anciennes versions et archives de travail conservées pour référence

## Démarrage

1. Ouvrez une classe Java dans src/main/java
2. Compilez-la avec javac :
   ```bash
   javac -d out chemin/vers/Fichier.java
   ```
3. Exécutez la classe compilée :
   ```bash
   java -cp out NomDeLaClasse
   ```
4. Utilisez Playground.java comme zone de prototypage et de tests rapides

## Philosophie du projet

Ce dépôt suit une logique pédagogique rigoureuse : comprendre, pratiquer, puis consolider. L’objectif est de développer une méthode de travail claire, propre et professionnelle, conforme à un environnement académique sérieux.
