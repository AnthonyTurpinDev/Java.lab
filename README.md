# java.lab

Projet Java de laboratoire pour pratiquer l’algorithme, l’orienté objet, les structures de données et les design patterns.

## Structure

- src/main/java/lab/sandbox/Playground.java : zone de test libre
- src/main/java/lab/utils/Printer.java : affichage propre
- src/main/java/lab/utils/Timer.java : mesure de temps
- src/main/java/lab/exercises/algo/BubbleSort.java : exercice prêt à exécuter

## Démarrage

1. Compilez avec `javac`
2. Exécutez les classes Java depuis `src/main/java`
3. Utilisez `Playground.java` pour tester rapidement vos idées
